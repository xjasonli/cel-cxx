- [Encoders Extension](extensions/encoders.md) - Base64 and URL encoding/decoding
- [Lists Extension](extensions/lists.md) - Enhanced list operations
- [Math Extension](extensions/math.md) - Mathematical functions and operations
- [Protocol Buffers Extension](extensions/proto.md) - Protobuf message support
- [Regular Expression Extension](extensions/re.md) - Basic regex operations
- [Regex Extension](extensions/regex.md) - Advanced regex support
- [Sets Extension](extensions/sets.md) - Set operations and utilities
- [Strings Extension](extensions/strings.md) - Advanced string manipulation
